// THE CPMM FORMULA — HOW PRICE IS DETERMINED
// CONSTANT PRODUCT x × y = k
x = YES token reserves
y = NO token reserves
k = constant product (invariant)

Price YES = y / (x + y)
Price NO = x / (x + y)

Example: x=75, y=25, k=1875
Price YES = 25/(75+25) = 0.25 → 75% prob
Price NO = 75/(75+25) = 0.75 → 25% prob

After buy of 10 YES:
new x = 75+10 = 85
new y = k/85 = 22.06
new Price YES = 22.06/107.06 = 0.206
